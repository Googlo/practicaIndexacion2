Consejos para la consulta
Utilice este formulario para buscar documentos de este Web que contengan palabras o combinaciones de palabras específicas. El sistema de búsqueda de texto mostrará una amplia lista de documentos que las contengan, con las mayores coincidencias en primer lugar.
Cada elemento de la lista es un vínculo al documento que contiene el texto buscado, si este documento tiene título lo mostrará, en caso contrario aparecerá el nombre del archivo. Además, podrá ver las coincidencias señaladas de dos formas: en modo resumen, y a texto completo.

Lenguaje de Consulta

El sistema de búsqueda de texto permite hacer consultas en formato de expresiones booleanas arbitrarias con las palabras clave AND, OR y NOT, y agrupadas entre paréntesis. Por ejemplo:

información recuperación
busca documentos que contengan 'información' o 'recuperación'

información or recuperación
lo mismo que arriba

información and recuperación
busca documentos que contengan ambos 'información' y 'recuperación'

información not recuperación
busca documentos que contengan 'información' pero no 'recuperación'

(información not recuperación) and WAIS
busca documentos que contengan 'WAIS', más 'información' pero no 'recuperación'

web*
busca documentos que contengan palabras que comiencen con 'web'
