Consejos para una búsqueda más eficaz

Simplificando, se pueden realizar consultas de una sola palabra o una frase completa. Sin embargo, con los consejos que encontrará en esta página, puede ampliar su consulta para obtener resultados más completos. Estos consejos le ayudarán a comprender la base del lenguaje de consulta que el potente buscador a texto completo Microsoft Index Server utiliza.

  • Realice búsquedas de palabras con el mismo prefijo. Por ejemplo, introduza en el formulario de consulta cervant* para encontrar Cervantes, cervantino, cervantina, cervantista, etc.

  • Realice búsquedas de todas las formas de una misma palabra. Por ejemplo, teclee en el formualrio de búsqueda coser** para encontrar coser, cosiendo, , y cosido.

  • Es preferible que realice consultas con la palabra clave NEAR, en lugar de AND, para palabras que se encuentren cercanas y dentro de un mismo contexto.

  • Puede restringir aún más sus consultas con AND NOT para excluir una cadena de texto de su consulta. Por ejemplo, si desea encontrar todas las ocurrencias de Alonso pero no Alonso Quijano, escriba la siguiente conuslta:

    alonso AND NOT quijano

  • Añada la palabra clave OR para encontrar todas las ocurrencias de una palabra o la otra, por ejemplo:

    Quijote OR Quijano

    Esta consulta encontrará todas las páginas que hagan referencia a Quijano o a Quijotees o a ambos nombre.

  • Si desea que este motor de búsqueda procese sus consultas de forma literal enciérrelas entre comillas. Por ejemplo, si introduce la consulta siguiente:

    "don quijote de la mancha"

    Index Server buscará literalmente las ocurrencias de don quijote de la mancha. Pero si introduce la misma consulta sin las comillas:

    don quijote de la mancha

    Index Server buscará todos los documentos que contengan las palabras don , quijote, y mancha.

Estos consejos son una mera introducción. si desea saber más sobre consultas más complejas, vea Query Language page.
